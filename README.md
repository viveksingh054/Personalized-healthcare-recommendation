##  Overview
**healthrec** (short for “healthcare recommendation”) is a streamlined, portfolio-style project demonstrating how machine learning applied to sample patient data (in `blood.csv`) can yield personalized healthcare insights.  
This is for educational/demo use — *not medical advice*.
